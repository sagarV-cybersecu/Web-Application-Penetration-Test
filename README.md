# 🌐 Web Application Penetration Testing

## 📌 Overview
This project demonstrates a black-box web application penetration test conducted using OWASP methodologies.

---

## 🔍 Testing Scope

- Authentication testing  
- Session management  
- Input validation  
- Security headers  
- Endpoint analysis  

---

## 🛠️ Tools Used

- Burp Suite  
- Nmap  
- Browser DevTools  

---

## ⚔️ Testing Activities

- Intercepted HTTP requests using Burp Suite  
- Analyzed authentication mechanisms  
- Tested input fields for vulnerabilities  
- Reviewed cookies and session security  
- Identified misconfigurations  

---

## 🎯 Result

Evaluated application security posture and identified potential security risks based on OWASP guidelines.

---

## ⚠️ Disclaimer

Testing conducted for educational purposes only.
